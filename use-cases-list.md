# PinnAI — Use Cases pour page web

## Liste complète (18 use cases)

### Originaux (Benoit)

1. **Configurateur de vente — Meubles sur mesure**
   - Vendeurs configurent dimensions/matériaux/finitions via interface
   - Visualisation en temps réel + prix final auto
   - Vente sur tablette chez le client

2. **Configurateur de vente — Engrais**
   - Configuration formulation pour le client
   - Flux vente → production

3. **Agent créateur de projets — Layer IA sur Zoho**
   - Upload soumission PDF → questions clarification → création auto projet + tâches + heures

4. **Agent facilitateur de facturation**
   - Mémoire complète des détails par projet
   - Brouillons de factures auto, TDs
   - Multi-systèmes: Zoho, SAP, logiciels d'achat

5. **Agent Dashboard — Connecté à SAP**
   - Dashboard temps réel: santé projets, ressources, attribution

6. **Facilitateur de procédures internes & production**
   - Procédures → guides interactifs + automatisation
   - Ex: release SolidWorks

7. **Interface de production sur mesure**
   - Layer par-dessus SAP ou standalone
   - Exactement les bons paramètres à l'écran

8. **Intégration complète — Unifier le disparate**
   - Excel + SAP + logiciel usine → UN logiciel intégré
   - IA pour rapports auto, tâches répétitives

9. **Plateforme de procédures — Humain + Automatisation en boucle**
   - Mix: auto pour les étapes faciles, guidage humain pour les complexes
   - Avec ou sans IA, automatisation programmée intégrée

### Générés (brainstorm agent)

10. **Suivi des non-conformités & retours clients**
    - Photo + description vocale → fiche auto + historique correctifs
    - Ex: usinage, plastique

11. **Planificateur de maintenance préventive**
    - Heures machine, alertes, bons de travail auto
    - Ex: CNC, presses, convoyeurs

12. **Gestion des certifications & formations**
    - Certifications à jour, rappels, rapport d'audit en 30 sec
    - Ex: CNESST, MAPAQ

13. **Estimateur de coût de revient par produit**
    - Coût réel vs prix de vente = vraie marge
    - Ex: armoires, pièces usinées

14. **Assistant d'achat & gestion fournisseurs**
    - Centralise fournisseurs, compare prix/délais, bons de commande auto
    - Ex: armoires, plastique, alimentaire

15. **Suivi des temps réels par job**
    - Scan 5 sec sur tablette → heures par projet temps réel → facturation
    - Ex: métallerie, sous-traitance

16. **Portail client — Suivi commande & approbation**
    - Client voit avancement, approuve plans, signe changements
    - Ex: portes/fenêtres, escaliers, mobilier commercial

17. **Gestionnaire de recettes & formulations**
    - Recettes centralisées, feuille de production ajustée au lot
    - Ex: alimentaire, peinture, chimie

18. **Suivi chantier & livraisons (fab + installation)**
    - Synchro atelier ↔ terrain, alertes pièces prêtes
    - Ex: structures acier, armoires commerciales
